# 快速上手

```ts
import { ohmylive2dPlugin } from 'vuepress-plugin-oh-my-live2d';

export default defineUserConfig({
  plugins: [
    ohmylive2dPlugin({
      // 在这里进行配置
    })
  ]
  //  ...more
});
```

::: tip
本项目不提供任何 Live2D Model 的下载地址。该 Live2D 模型来自于网络，OhMyLive2D 默认配置了一个模型地址，方便在初次使用时体验预览效果，仅供参考和学习。
:::
